Super Dumb Player

A Flash video player which has no Flash user interface and no error handling.

Get the source and what not at the Github page.

Requirements

  1. jQuery
  2. jQuery UI with Draggable
  3. swfobject

UnFeatures

Does supports more than one video/player at a time. Probably only works with progressive video. "Tested" in IE 7, Safari, Chrome and Firefox 3.5, but not heavily yet. Doesn't show a buffering state.

Basic Usage

  1. Create an element to contain your video. It must have a unique id and should include height and width styles. This element and its contents will be replaced by the flash movie. If Flash it not available this element will be left alone.
  2. Create the player.

    vp = dumb_player.create("my_unique_id", "vide_file_path.m4v");

Custom Templates & Using Canvas

The player does have a built in HTML interface. Use CSS to style the ui. Each element in the UI requires a specific class name, see below.

This template can be replaced using dumb_player.ui.template(template_source). You must replace the template before you create the player.

The UI does not need to have all, or any, of the elements though. No errors should occur if something is missing.

It is possible to use a graphics library, like Raphael, to really customize the look. 


      <div class='dump_player_ui'>
        <span class='duration'/>
        <span class='time'/>
        <div class='scrubber'>
          <div class='track'>
            <div class='buffer'>
              <div class='thumb'/>
            </div>
          </div>
        </div>
        <button class='state'>
          <span class='play'>Pause</span>
          <span class='pause'>Play</span>
        </button>
        <button class='volume'>
          <span class='on'>Mute</span>
          <span class='off'>Unmute</span>
        </button>
      </div>

API

  • dumb_player.player_source

    Set this property to point to the URL for the super_dumb_player.swf

  • dumb_player.create(target_element, video_src) ⇒ dumb_player instance

    Creates a new player instance. Playing is always auto.

  • dumb_player.ui.template() ⇒ template source

    dumb_player.ui.template(template_source) ⇒ template source

    This is optional. Set or retrieves the html template to use for the UI. A default UI will be used if you do not set one.

  • dumb_player.ui.time_formater(function)

    This is optional. The time formatter is a function applied to the duration and time data. Use this function to manipulate the incoming decimal data for rounding.

Instance API

  • instance.player() ⇒ the object/embed element

    Return the object or emended element containing the Flash player.

  • instance.play() ⇒ instance

    Play the movie, if it is paused.

  • instance.pause() ⇒ instance

  • instance.load(src) ⇒ instance

    Load a new movie and start playing it

  • instance.remove()

    Remove the player and replace it with the original content.

  • instance.toggle() ⇒ instance

    Toggle play/pause.

  • instance.toggle_volume() ⇒ instance

    Toggle volume on or off.
